## 测试用例

### M-1 记忆单元建模 — 内容字段

**说明**：`t_memory` 表定义了 content / summary / key_points 三个内容层次。

**代码：** `app/models/base.py:170-171`

```python
content = Column(Text, nullable=False)       # 记忆核心内容
summary = Column(Text)                       # 记忆摘要
key_points = Column(JSON, default=list)      # 关键要点列表
```

**验证**：写一条记忆，查 `t_memory` 表确认 `content` 有值。

memory_type = Column(String(64), nullable=False, index=True,

### M-5 任务状态管理

**请求 — 创建任务：**
```json
POST /api/v1/task
{
  "user_id": "u_task",
  "title": "编写技术方案",
  "goal": "完成Q3技术方案文档"
}
```

**预期：** `task_id = task_xxx`, `status = "pending"`

**代码：** `app/api/v1/task.py:53-90`

---

**请求 — 更新进展：**
```json
PUT /api/v1/task/task_xxx
{
  "status": "in_progress",
  "progress": "已完成需求分析，正在设计方案",
  "completed_items": ["需求文档"],
  "pending_items": ["技术方案", "代码实现"]
}
```

**预期：** `status = "in_progress"`, `completed_items` / `pending_items` 更新

**代码：** `app/api/v1/task.py:174-217`

---

**请求 — 非法状态转换（应被拒绝）：**
```json
PUT /api/v1/task/task_xxx
{"status": "pending"}
```
（当前已是 `in_progress`，不能直接回退到 `pending`）

**预期：** `code=-1`, `error_code: CONFLICT_ERROR`

**代码：** `app/api/v1/task.py:41-50` — `_validate_status_transition`

---

**请求 — 完成任务：**
```json
POST /api/v1/task/task_xxx/complete
```

**预期：** `status = "completed"`, `ended_at` 有值

**代码：** `app/api/v1/task.py:261-285`

```python
"""测试：通用记忆建模与多层记忆管理"""
import httpx, json
BASE = "http://localhost:8000/api/v1"
OK, FAIL = 0, 0
def check(label, condition, detail=""):
    global OK, FAIL
    if condition: OK += 1; print(f"  PASS: {label}")
    else: FAIL += 1; print(f"  FAIL: {label} {detail}")

# === M-1/M-2/M-4: 记忆单元建模 ===
print("=== M: 记忆建模 ===\n")
r = httpx.post(f"{BASE}/memory/write", json={
    "user_id":"u_model","session_id":"sess_model",
    "messages":[{"role":"user","content":"我叫张三，后端工程师"}],
}, timeout=10)
check("M-1 写入成功", r.status_code == 200)
results = r.json()["data"].get("results",[])
if results:
    check("M-2 memory_type", "memory_type" in str(results[0]))
    check("M-4 用户标识", "u_model" in str(r.json()))

# === M-5: 任务管理 ===
print("\n=== M-5: 任务管理 ===\n")
r = httpx.post(f"{BASE}/task", json={
    "user_id":"u_task","title":"测试任务","goal":"验证任务功能",
}, timeout=10)
task_id = r.json()["data"]["task_id"]
check("创建任务200", r.status_code == 200, f"或201, 实际{r.status_code}")
check("task_id以task_开头", task_id.startswith("task_"))
check("初始status=pending", r.json()["data"]["status"] == "pending")

# 更新进展
r = httpx.put(f"{BASE}/task/{task_id}", json={
    "status":"in_progress","progress":"进行中",
    "completed_items":["A"],"pending_items":["B"],
}, timeout=10)
check("更新进展200", r.status_code == 200)
check("状态变为in_progress", r.json()["data"]["status"] == "in_progress")

# 非法转换
r = httpx.put(f"{BASE}/task/{task_id}", json={"status":"pending"}, timeout=10)
check("非法转换拒绝", r.json()["code"] != 0, f"code={r.json().get('code')}")

# 完成
r = httpx.post(f"{BASE}/task/{task_id}/complete", timeout=10)
check("完成任务200", r.status_code == 200)

# 进展摘要
r = httpx.get(f"{BASE}/task/{task_id}/progress", timeout=10)
check("进展摘要200", r.status_code == 200)
check("related_memory_count字段", "related_memory_count" in r.json()["data"])

# === L-1/L-2/L-3: 多层记忆 ===
print("\n=== L: 多层记忆 ===\n")
# 会话级
for text in ["第一次见面","今天心情好"]:
    httpx.post(f"{BASE}/memory/write", json={
        "user_id":"u_layer","session_id":"sess_layer",
        "messages":[{"role":"user","content":text}],
    }, timeout=10)
check("L-2 会话记忆写入2条", True)

# 用户级检索
r = httpx.post(f"{BASE}/memory/search", json={
    "query":"偏好","user_id":"u_layer","top_k":5}, timeout=10)
check("L-1 用户级检索200", r.status_code == 200)

print(f"\n=== 结果: {OK} PASS / {FAIL} FAIL ===")
```
